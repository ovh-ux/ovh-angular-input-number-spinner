inputNumberSpinner
directive in module inputNumberSpinner

Description

Enhance a number input for entering numeric values, with up/down buttons and arrow key handling.

Usage

as element:
<input-number-spinner
       ng-model="{Number}"
       [input-number-spinner-min="{Number}"]
       [input-number-spinner-max="{Number}"]
       [input-number-spinner-step="{Number}"]
       [input-number-spinner-button-class="{String}"]
       [input-number-spinner-acceleration-speed="{Number}"]
       [input-number-spinner-time-before-acceleration="{Number}"]
       [input-number-spinner-on-change="{Function}"]
       [ng-readonly="{Boolean}"]>
</input-number-spinner>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ng-modelNumber

The model of the input

input-number-spinner-min
(optional)
Number

The minimum value of the input

(default: -Infinity)

input-number-spinner-max
(optional)
Number

The maximum value of the input

(default: Infinity)

input-number-spinner-step
(optional)
Number

The size of the step to take when spinning

(default: 1)

input-number-spinner-button-class
(optional)
String

The bootstrap class to add to buttons. For example : if buttonClass is equal to primary the buttons will have btn-primary class

(default: default)

input-number-spinner-acceleration-speed
(optional)
Number

Time (in ms) between two interval

(default: 75)

input-number-spinner-time-before-acceleration
(optional)
Number

Time (in ms) before interval is launched

(default: 500)

input-number-spinner-on-change
(optional)
Function

Callback function called when value has changed

ng-readonly
(optional)
Boolean

Set component read-only

(default: false)
